# プロジェクト概要

> Last Updated: 2026-03-17

## ゴール
LSTEPの依存関係を把握し、ユーザーが新機能追加・設定変更を指示した際に、依存関係の影響を踏まえてミスなく実行できる半自動化環境を構築する。

## マイルストーン

| マイルストーン | ステータス | 備考 |
|---|---|---|
| M1: 現状把握（F1-F3 全設定読み取り） | 🔨 進行中 | Phase 0〜2 に相当 |
| M2: 依存関係の可視化（F4 Mermaid図） | ⬜ 未着手 | Phase 3 に相当 |
| M3: 半自動化（指示→依存チェック→実行） | ⬜ 未着手 | Phase 4 に相当 |
