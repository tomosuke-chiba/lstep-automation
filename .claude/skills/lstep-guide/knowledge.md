# Lstep ナレッジベース

> 最終更新: 2026-03-21 | データソース: data/*.json

## 概要

矯正歯科のLINE公式アカウント。Lステップで自動配信・タグ管理・依存関係を構築。
ゴール: 友だち登録 → 初診予約 → 精密検査 → 契約 を自動配信で支援。

## 変更時のチェックリスト

| 変更したいもの | 確認すべきこと |
|---|---|
| テンプレートの内容変更 | そのテンプレートを使う全シナリオ（上記エッジ一覧参照） |
| タグ名の変更・削除 | そのタグを発動条件にしているシナリオ・自動応答 |
| 友だち情報フィールド名変更 | シナリオ内の変数参照 {友だち情報:フィールド名} |
| シナリオの停止・削除 | 他シナリオからの「シナリオ開始」アクション参照 |
| 新しいシナリオ追加 | 既存タグ・テンプレートとの命名衝突 |

## データファイルの場所

最新データは data/ ディレクトリ内のタイムスタンプ付きJSON。
- `data/scenarios_*.json` — シナリオ一覧
- `data/tags_*.json` — タグ一覧（フォルダ・友だち数含む）
- `data/friend_fields_*.json` — 友だち情報フィールド一覧
- `data/dependency_graph_*.json` — 依存関係グラフ（nodes + edges）
- `data/scenario_details_*.json` — シナリオ詳細（ステップ・テンプレート参照）
- `data/mermaid_*.md` — Mermaid図
- `docs/system-overview.html` — 人間向け全体構成マップ（ブラウザで閲覧）

分析ツール: `src/planner.ts` の analyze() / searchNodes() / checkDataFreshness()
